An Unofficial EAT THE REICH System for Foundry VTT

EAT THE REICH is a chaotic, ultra-violet tabletop RPG where players take on the roles of vampiric commandos on a bloody mission to tear through Nazi-occupied France during World War II. Written by Grant Howitt and illustrated by Will Kirkby, and published by Rowan, Rook and Decard. The game blends pulpy action, dark humor, and horror as players use their supernatural abilities to slaughter Nazis and consume their blood. With fast-paced mechanics and a rebellious tone, EAT THE REICH delivers an over-the-top, cathartic power fantasy drenched in gore and irreverence.

Find the books here: EAT THE REICH

This Eat the Reich Package for Foundry VTT is an independent production by Joseph Hopson (ephson) and is not affiliated with Rowan, Rook and Decard or Will Kirkby. It is published under the RR&D Community License.

How to Install

You can install the latest released version of the system by using this manifest link in Foundry VTT. Instructions: https://github.com/philote/eat-the-reich/releases/latest/download/system.json

Features

  • Character Sheet
  • NPC/Threat Sheet
  • Location Sheet
  • Item Sheets for [advance, bonus, description, equipment, objective]
  • Basic Dice Pool support
    • Dice can be allocated in chat
  • Supports light and dark modes
